# Reviews — 周 / 月 / 季度复盘

### 节奏

- **周复盘**:`YYYY-WW.md`(例如 `2026-W16.md`)— 每周一次,15-20 分钟
- **月复盘**:`YYYY-MM.md`— 每月一次,30-45 分钟
- **季度复盘**:`YYYY-QQ.md`— 每季度一次,60-90 分钟

### 周复盘模板(最小版本)

```markdown
# YYYY-Www

## 事实(3 件本周发生的)

## 一个惊讶

## 一个坑

## 一个小胜

## 下周一件事
(不是 list,就一件)

## Claude 留记号
(Claude 对照 patterns.md 看本周有没有某个 pattern 重演,在这里记)
```

### 月 / 季度复盘

月复盘看 4 周 journal + 4 周周复盘 → 找 pattern。
季度复盘看 3 个月月复盘 + goals.md → 检查方向是否还对齐。

### 原则

- **不是周报**。不列任务清单。
- 每次复盘让**用户自己**说一句"这次我学到的是 ___",不让 Claude 代劳。
